API Documentation

Explore our RESTful API endpoints for managing todos and messages.

Message API Endpoints

GET /api/messages
Get all messages
GET /api/messages/{id}
Get a message by ID
POST /api/messages
Create a new message

Todo API Endpoints

GET /api/todos
Get all todos
GET /api/todos/{id}
Get a todo by ID
POST /api/todos
Create a new todo
PUT /api/todos/{id}
Update a todo
DELETE /api/todos/{id}
Delete a todo
